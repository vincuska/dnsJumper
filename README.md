# dnsJumper
[dnsJumper](https://www.sordum.org/7952/dns-jumper-v2/) remade in Python

## Features

When running it will test all the hosts from the .txt file and order them by fastest.

## Run Locally

```batch
  git clone https://github.com/vincuska/dnsJumper
  cd dnsJumper
```

```batch
  python main.py
```
